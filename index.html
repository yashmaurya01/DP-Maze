About This Game

In this DP Honeycomb Maze, you navigate a 4x4 grid. Some doors lead to water (loss), some are locked, and guards roam inside! Your goal: move from the top row to the bottom without getting caught or drowned.

You can ask DP queries about the doors before choosing one. Each query consumes budget and may return a noisy (but differentially private) answer. Adjust the epsilon to control how accurate answers are. Higher epsilon = more accurate answers.

How Differential Privacy Works Here

Each query reduces your privacy budget. The query cost sets how much budget you spend per query. Epsilon sets the DP parameter: higher epsilon = less noise = more accurate answers.

By tuning these parameters, you control the trade-off between number of queries and accuracy. DP ensures you can't perfectly learn the maze and guard placement with certainty.

Created as a demonstration of interactive DP in a maze game. Neon orange & black theme.
